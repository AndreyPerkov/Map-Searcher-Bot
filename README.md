# Map Searcher Bot for Discord

Bot version : **1.1**

a Python bot for Discord, which searches for maps by their name and etc!

# How to use this bot?

1. install all libraries with `pip install -r requirements.txt`
2. Edit `config.json` file.
3. Run the bot.
